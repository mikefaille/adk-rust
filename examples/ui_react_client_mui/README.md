# UI React Client (MUI)

This is the **Enterprise/Themed** alternative to the standard React client, built with Material UI (MUI). It provides a polished, professional interface for interacting with ADK agents.

## Setup

### Prerequisites

- Node.js 18+

### Installation

```bash
npm install
```

### Running Development Server

This client runs on port **3001** to avoid conflicts with the standard client.

```bash
npm run dev
```

## Configuration

Copy `.env.example` to `.env` and adjust if necessary:

```bash
cp .env.example .env
```
